# Omani Bread Store

## Project Description
This project is an online store for Omani bread, aiming to provide a platform for ordering and delivering freshly baked traditional bread across Oman.

## Features
- User-friendly interface for browsing products.
- Simple ordering and checkout process.
- Real-time order tracking.
- Integration with WhatsApp for customer inquiries.
- GitHub Pages deployment for seamless updates.

## How to Use
1. Visit the website.
2. Browse the products available.
3. Add desired items to your cart.
4. Proceed to checkout to place your order.

## Installation Instructions
To run this project locally, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/Aldarmaki1000/omani-bread-store.git
   ```
2. Navigate to the project directory:
   ```bash
   cd omani-bread-store
   ```
3. Install necessary dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm start
   ```

## GitHub Pages Deployment Guide
Follow these steps to deploy the project to GitHub Pages:
1. Ensure your code is pushed to the `main` branch.
2. Go to the repository settings on GitHub.
3. Under "Pages", select the branch to deploy (e.g., `main`).
4. Set the folder to `/root` and save.

## WhatsApp Integration Details
To integrate WhatsApp:
- Add the following code snippet to the website for enabling direct message link:
   ```html
   <a href="https://wa.me/YOUR_PHONE_NUMBER">Chat with us on WhatsApp</a>
   ```
Replace `YOUR_PHONE_NUMBER` with your actual WhatsApp number.

## Customization Guide
Users can customize the store by:
- Modifying the product listings in the `data/products.js` file.
- Changing styles in the `styles.css` file.
- Updating the text content in the relevant HTML/PHP files.
